# 這裡放置各種C語言的演算法
你可以找到你需要的演算法，下載它，然後在你的專案中使用它，**不需要**任何授權費用，完全免費。**但不可聲明這些演算法是你寫的，或者是你專利的，因為這些演算法都是開源的，任何人都可以使用。**

你也可以**貢獻**你的算法，或者**改進**現有的算法，讓它們更快，更準確，更好用。你可以提交pull request，或者直接聯繫我，我會把你的算法加入到這個專案中
### 但必須遵守以下規則：
資料夾名：
`中文名+英文名` **(英文使用大駝峰式)**
如：`數學式解析MathematicalAnalysis`

檔名則隨便，但必須是C語言的檔案，且副檔名為`.c`或`.h`

若是你想要寫的算法已經有類似的標題了，那麼資料夾名稱須加上`-`編號
如：`數學式解析MathematicalAnalysis-2`

# This is a collection of various C language algorithms.

You can find the algorithm you need, download it, and use it in your projects. **No** licensing fees are required; it's completely free. **However, you cannot claim that these algorithms were written by you or are your patents, as they are all open-source and can be used by anyone.**

You can also **contribute** your own algorithms or **improve** existing ones to make them faster, more accurate, and easier to use. You can submit a pull request or contact me directly, and I will add your algorithm to this project.

### However, the following rules must be followed:

Folder Name:

`Chinese Name + English Name` **(Use Camelhamx for the English name)**

For example: `數學式解析MathematicalAnalysis`

Filename: This is flexible, but the file must be a C language file with the extension `.c` or `.h`

If your algorithm already has a similar title, then the folder name must include a `-` number.

For example: `數學式解析MathematicalAnalysis-2`
